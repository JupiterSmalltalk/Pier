Welcome to Pier

The Ultimate Content Management System

Pier is a powerful and extensible implementation of a
meta-described content management and Wiki system.

Written with objects from top to bottom.

Object-Oriented Design

Pier is an object oriented, meta-described domain model built using Magritte. The content of the pages is parsed and stored as a tree of entities representing everything on your site.

Extensibility

Everything in Pier can be extended - page types, storage mechanism, actions, security, the web server - everything. Plug-ins can be shared and loaded independently of each other.

Open Source

Pier is released under the MIT license which grants unrestricted rights to copy, modify, and redistribute as long as the original copyright and license terms are retained.

Test Suites

Pier has more than 1200 unit tests. This makes it easy to change and verify the code and comes in extremely useful when porting Pier to other Smalltalk dialects or when writing extensions.
